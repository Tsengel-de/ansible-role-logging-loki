# Ansible Role: logging-loki

Minimal Ansible role for Loki logging configuration.

## Description

This role provides basic Loki setup for log aggregation.

## Requirements

- Ansible 2.9+
- Debian/Ubuntu or RHEL/CentOS systems

## Example Playbook

```yaml
- hosts: servers
  roles:
    - Tsengel-de.logging-loki
```
